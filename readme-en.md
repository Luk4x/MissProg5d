## 📝 About

> Watching the video above and/or accessing the project online will help you understand the explanation!

The project consists of a list of tasks developed with HTML, CSS and Javascript, where it is possible to add a task, mark it as completed and remove it.<br>
Tasks are also saved in `localStorage` to improve the user experience.

### 📌 Some Highlights

- Use of `localStorage`;
- Small mapping of the `Enter` key to add a task;
- Responsiveness

## 📖 Cloning the Project

To clone and run this project on your computer, you just need [Git](https://git-scm.com/) to be previously installed.<br>
After that, in the terminal:

```bash
# Clone this repository with:
> git clone https://github.com/Luk4x/MissProg5d.git

# Enter the repository with:
> cd MissProg5d

# Run the project with:
> start index.html # For Windows users
> open index.html # For Linux/Mac users
```
